# Sample 3D JS

* scanline rasterizer
* raycaster (wolfenstein3D style)
* voxels

## Raycaster keys

* wasd: movement
* q: turn left
* e: turn right
* 1: only lightmaps
* 2: only textures

## Assets used

* African head and Diablo: https://github.com/ssloy/tinyrenderer
